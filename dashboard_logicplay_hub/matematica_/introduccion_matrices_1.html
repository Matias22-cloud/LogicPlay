¿Qué es una matriz?

Imagina una tabla de datos organizada. Una matriz es un conjunto de números dispuestos en **filas** (horizontales) y **columnas** (verticales). En los videojuegos, se usan para posicionar personajes en un espacio 3D.

Regla de Oro: La dimensión se define como Filas (m) x Columnas (n).
Estructura Básica
[ a11   a12 ]
[ a21   a22 ]

Ejemplo de Identificación

Analicemos la matriz B:

B = [ 10   20   30 ]
Filas: 1 (Horizontal)
Columnas: 3 (Verticales)

Dimensión Total: 1x3
