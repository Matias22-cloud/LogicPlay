Lanzamiento de proyectil horizontal

help Enunciado

Un avión de rescate vuela horizontalmente a una altura de 125 metros con una velocidad constante de 40 m/s. Se deja caer un paquete de suministros. ¿A qué distancia horizontal del punto de lanzamiento caerá el paquete? (Considere g = 9.8 m/s²)

calculate Procedimiento Algebraico

1. Análisis Vertical (Y)

Calculamos el tiempo de caída (t) usando:

y = v0yt + ½gt²

Como el lanzamiento es horizontal, v0y = 0

125 = 0 + ½(9.8)t²
125 = 4.9t²
t² = 125 / 4.9
t² ≈ 25.51
t ≈ 5.05 s

2. Análisis Horizontal (X)

Calculamos el alcance (R) usando el tiempo obtenido:

x = vxt

La velocidad horizontal es constante:

x = (40 m/s) * (5.05 s)
x = 202 metros
task_alt

Solución Final

El paquete caerá a una distancia horizontal de 202 metros.

2

Lanzamiento con ángulo

help Enunciado

Un futbolista patea un balón con una velocidad inicial de 20 m/s y un ángulo de 30° respecto a la horizontal. Calcule la altura máxima que alcanza el balón. (Considere g = 10 m/s²)

calculate Procedimiento Algebraico

Descomposición Vectorial y Altura Máxima

1. Primero descomponemos la velocidad inicial en su componente vertical (v0y):

v0y = v0 * sen(θ)
v0y = 20 * sen(30°)
v0y = 20 * 0.5 = 10 m/s

2. En la altura máxima, la velocidad vertical final (vfy) es igual a 0. Usamos la fórmula:

vfy² = v0y² - 2gh
0² = (10)² - 2(10)h
0 = 100 - 20h

3. Despejamos la altura (h):

20h = 100
h = 100 / 20
h = 5 metros
task_alt

Solución Final

La altura máxima alcanzada por el balón es de 5 metros.
