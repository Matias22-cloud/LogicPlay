Método de Gauss-Jordan

El objetivo: Transformar una matriz aumentada hasta que la parte izquierda sea una **Matriz Identidad**. Los valores resultantes a la derecha serán las soluciones de tus incógnitas.

Ejemplo Resuelto por Mí: Sistema 2x2

Ecuaciones: x + 2y = 8 y 0x + y = 3

[ 1   2 | 8 ]
[ 0   1 | 3 ]
F1 = F1 - 2*F2 trending_flat
[ 1   0 | 2 ]
[ 0   1 | 3 ]

Solución final: x = 2, y = 3

Desafío: Operación de Filas

Si tenemos la fila F1 = [ 2   4   10 ] y queremos que el primer número sea 1, ¿por qué número debemos dividir toda la fila?
