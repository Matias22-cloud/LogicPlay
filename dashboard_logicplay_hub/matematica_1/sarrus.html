Álgebra Lineal

La Regla de Sarrus

¿Qué es?

Es un método que solo funciona para matrices de **3x3**. Su objetivo es calcular el determinante de forma rápida. El truco visual consiste en imaginar que las dos primeras filas se repiten debajo de la matriz original para completar las "diagonales".

| a₁ b₁ c₁ |
| a₂ b₂ c₂ |
| a₃ b₃ c₃ |
| a₁ b₁ c₁ |
| a₂ b₂ c₂ |

Ejemplo Resuelto por Mí

Calcular el determinante de la matriz M:

M = [ 3   2   1 ] / [ 0   2   2 ] / [ 1   0   1 ]
1

Diagonales Principales (Sumar):
(3·2·1) + (0·0·1) + (1·2·2) = 6 + 0 + 4 = 10

2

Diagonales Secundarias (Restar):
(1·2·1) + (2·0·3) + (1·2·0) = 2 + 0 + 0 = 2

Determinante = 10 - 2 = 8
